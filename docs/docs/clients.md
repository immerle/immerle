# Connecting clients

Immerle implements the **Subsonic / OpenSubsonic** API, so any compatible client
works out of the box.

## Set it up

Point your client at your server and sign in with the account you created during
[installation](./installation.md):

- **Server / URL:** `http://<host>:4533`
- **Username / Password:** your Immerle credentials

## Tested clients

- [Supersonic](https://github.com/dweymouth/supersonic) (desktop)
- [Symfonium](https://symfonium.app/) (Android)
- [DSub](https://github.com/daneren2005/Subsonic) (Android)

Any other Subsonic/OpenSubsonic client should work too. If something doesn't,
it's worth [opening an issue](https://github.com/immerle/immerle/issues).
